Here are some of my projects!

Basic Circuitry for Kids!

Small interactive lessons teach the basics of circuitry. The web interface interacts with an Arduino board to produce feedback through lights.

Home-ing Network

Designed to crowdsource information on the location of homeless people around Atlanta.

Mock Blog

A responsive, functional framework for an image-led blog or website (MEN stack). Styled with Semantic UI and masonry.js (for a cascading grid of images.)

RGB Color Guessing Game

A simple javascript-run game where you guess which square has the correct rgb value.

City Select

A city review site that helps users better plan their next vacation or move. Uses a MEN stack, RESTful routing, vanilla JS, and more.

Tapapat

An interactive sound and animation kit inspired by Patatap. Utilizes paper.js and howler.js. (No mobile support yet)

Twitch TV Channel Viewer

This webpage renders the current status of some of the most popular twitch.tv channels with the Twitch API, vanilla JS, and Bootstrap.

Photo Gallery Webpage

Simple website for displaying photographs that I've taken and images I've edited.

To-Do List

My version of the infamous, low-functionality to-do list ;). It focuses mostly on CSS styling with jQuery events.
